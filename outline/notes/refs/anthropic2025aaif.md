# Donating MCP and Establishing the Agentic AI Foundation

## Overview

On December 9, 2025, Anthropic announced the donation of the Model Context Protocol (MCP) to the Agentic AI Foundation (AAIF), a new directed fund under the Linux Foundation. This marked the culmination of MCP's journey from internal Anthropic project to industry-standard protocol—exactly one year after its initial open-source release in November 2024.

The AAIF was co-founded by Anthropic, Block, and OpenAI, with support from Google, Microsoft, AWS, Cloudflare, and Bloomberg as platinum members. This unprecedented coalition of competing frontier model providers united around shared infrastructure signals a fundamental shift in how the AI industry approaches standardization.

## The Foundation

### Governance Structure

The AAIF operates as a "directed fund" under the Linux Foundation, leveraging decades of experience stewarding critical open-source projects (Linux kernel, Kubernetes, Node.js, etc.). Key governance principles include:

- **Vendor Neutrality**: No single company dominates; decisions made collaboratively
- **Open Governance**: Transparent decision-making with broad contributor input
- **Project Autonomy**: Individual projects (MCP, goose, AGENTS.md) maintain technical independence while benefiting from shared resources

The AAIF Governing Board handles strategic investments, budget allocation, member recruitment, and approval of new projects. Individual projects retain full autonomy over their technical direction and day-to-day operations.

### Founding Projects

Three projects anchor the AAIF at launch:

1. **Model Context Protocol (MCP)** — Anthropic
   - Universal standard for connecting AI models to tools, data, and applications
   - Transforms the N×M integration problem into N+M
   - By December 2025: adopted by ChatGPT, Cursor, Gemini, VS Code, Microsoft Copilot

2. **goose** — Block
   - Open-source, local-first AI agent framework
   - Combines language models, extensible tools, and MCP-based integration
   - Thousands of Block engineers use it weekly for coding, data analysis, and documentation

3. **AGENTS.md** — OpenAI
   - Simple, universal standard for providing AI coding agents with project-specific guidance
   - Adopted by 60,000+ open-source projects and major agent frameworks (Cursor, Devin, GitHub Copilot, Jules, VS Code)
   - Ensures consistent agent behavior across different repositories and toolchains

## MCP's Journey to Foundation Status

### One Year of Growth

From November 2024 to December 2025, MCP achieved remarkable adoption:

**Platform Adoption:**
- ChatGPT (OpenAI's consumer product)
- Cursor (AI-powered code editor)
- Gemini (Google's AI assistant)
- Microsoft Copilot
- Visual Studio Code
- Numerous other AI products and agents

**Enterprise Infrastructure:**
- Deployment support from AWS, Cloudflare, Google Cloud, and Microsoft Azure
- Block's internal deployment (thousands of engineers)
- Enterprise-grade production deployments

**Ecosystem Growth:**
- 16,000+ MCP servers created
- 97+ million monthly SDK downloads
- Claude connector directory with 75+ connectors
- Official community-driven Registry for discovering MCP servers
- SDKs in Python, TypeScript, and other major languages

**Technical Evolution:**
- November 2025 spec release: asynchronous operations, statelessness, server identity, official extensions
- Tool Search and Programmatic Tool Calling capabilities
- OAuth 2.1 authorization
- JSON-RPC batching for efficiency

### The Decision to Donate

Mike Krieger, Anthropic's Chief Product Officer, explained the rationale: "MCP started as an internal project to solve a problem our own teams were facing. When we open sourced it in November 2024, we hoped other developers would find it as useful as we did. A year later, it's become the industry standard for connecting AI systems to data and tools... Donating MCP to the Linux Foundation as part of the AAIF ensures it stays open, neutral, and community-driven as it becomes critical infrastructure for AI."

David Soria Parra, MCP co-creator, articulated the goal: "The main goal is to have enough adoption in the world that it's the de facto standard. We're all better off if we have an open integration center where you can build something once as a developer and use it across any client."

## Industry Significance

### From Competition to Collaboration

The AAIF represents a remarkable alignment among competitors:

- OpenAI (ChatGPT, GPT-4, o-series) partnering with Anthropic (Claude)
- All major cloud providers (AWS, Google Cloud, Azure) supporting shared infrastructure
- Developer tools companies, enterprise software vendors, and startups united

Jim Zemlin, Linux Foundation executive director, stated the vision: avoid "closed wall" proprietary stacks where tool connections, agent behavior, and orchestration are locked behind handful of platforms.

### The Web Analogy

Block's announcement drew explicit parallels to web history: "The Internet, Linux, and the Web succeeded precisely because they were open. They empowered anyone with talent and determination to build, innovate, and create value on top of open infrastructure. Agentic AI deserves the same opportunity."

Block hopes AAIF "can become what the W3C is for the Web: a set of standards and protocols that guarantee interoperability, open access, and freedom of choice with open source reference implementations."

### Enterprise Implications

For enterprise adoption, the foundation provides:

1. **Reduced Integration Risk**: Protocol backed by neutral foundation, not single vendor
2. **Investment Confidence**: Long-term viability assured by multi-stakeholder governance
3. **Interoperability Guarantees**: Standards developed collaboratively, not dictated

As one enterprise analysis noted: "With MCP operating under Linux Foundation stewardship, CIOs, enterprise architects, and systems integrators can treat the protocol more like other open infrastructure standards."

## Technical Continuity

The MCP governance model established earlier in 2025 continues unchanged under AAIF:

- Core maintainers retain technical direction
- Community-driven RFC process for protocol changes
- Anthropic continues active development and investment
- New features continue shipping (Tool Search, Programmatic Tool Calling, spec extensions)

As the official MCP blog stated: "For MCP little changes. The governance model we introduced earlier this year continues as is."

## Significance for Our Paper

The AAIF formation represents the maturation of agentic AI from experimentation to foundational infrastructure. For our paper's thesis about "HPC in the Agentic Era," this has several implications:

### Standards-Based Integration

HPC centers evaluating AI-assisted workflows can now build on stable, vendor-neutral protocols. MCP isn't going away, being deprecated, or getting fragmented by vendor forks—it has achieved the status of shared infrastructure akin to HTTP or SSH.

### Enterprise-Ready Governance

The Linux Foundation's governance model provides the kind of stability and transparency that research computing organizations require. HPC centers can confidently invest in MCP-based tooling knowing the standard has long-term viability backed by multi-stakeholder commitment.

### Ecosystem Benefits

The coalition supporting AAIF includes every major cloud provider and AI company. This means:
- First-class MCP support in all major AI assistants (Claude, ChatGPT, Gemini, Copilot)
- Enterprise deployment patterns from AWS, Google Cloud, Azure
- Growing ecosystem of servers, SDKs, and tooling
- AGENTS.md as a complementary standard for providing project-specific context

### The Path Forward for HPC

For HPC, the implication is clear: MCP servers for HPC-specific tools (scheduler interfaces, module systems, data management, documentation) represent the path to AI integration. Building on AAIF standards ensures interoperability with whatever AI assistants users prefer—whether they're using Claude in Warp, ChatGPT Desktop, or Cursor.

The parallel to the web era is instructive: just as web standards enabled any browser to work with any server, MCP enables any AI assistant to work with any tool server. HPC centers that build MCP servers for their systems are investing in durable infrastructure, not vendor-specific integrations.

The AAIF announcement caps our paper's timeline (2017-2025):

- **2017**: Transformers architecture (Vaswani et al.)
