# Abstract Workshop: "Not Merely Tolerated" (FB-P2.01 / Theme 15)

## Summary

Workshopped five alternatives for the awkward "not merely tolerated" phrasing in the abstract. Geoffrey chose Option D: "researchers expect these tools to be supported, not just permitted."

## Work Completed

Presented five options for replacing the sentence:
- **A**: Drop negation entirely ("work seamlessly within their existing workflows")
- **B**: Flip to positive demand ("want them to work on day one")
- **C**: Reframe as reality ("already use these tools and expect their computing environment to support them")
- **D**: Surgical fix — keep contrastive structure ("expect these tools to be supported, not just permitted") ← **selected**
- **E**: Recast as question ("how do you support a tool that can do your job?")

Geoffrey selected D for its clean rhythm and minimal disruption. The change swaps both the verb ("want" → "expect") and the contrast ("integrated/tolerated" → "supported/permitted"), producing a smoother read while preserving the original intent.
